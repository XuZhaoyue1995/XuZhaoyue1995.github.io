在paraview上录制脚本的方法
发布于：2022-04-15 15:28:46 标签：/ paraview / 访问：

有时候我们需要进行批量后处理或重复多次进行某一指令，这时候我们需要运行脚本来实现这一功能。Python和paraview有多种交互功能，其中脚本交互是最常见的一种。
最简单的运行方法为：通过点击paraview界面的 Tools > Start Trace 开始录制脚本。当进行所有你要重复执行的步骤后，选择
Tools > Stop Trace 结束脚本录制。这个时候，paraview会生成界面交互操作的python脚本。将该脚本进行保存。对其进行修改。再点击Macros > import new macro 导入保存的文件。此时会在Macros中生成一个名字相同的选项，点击该选项就能成功运行脚本。
